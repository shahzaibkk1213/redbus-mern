# RedBus MERN App
A full-stack bus ticket booking web app built with MongoDB, Express, React, Node.js.

## Features
- User Authentication with JWT
- Bus Search & Booking
- Payment Integration (Stripe)
- Bus Hire Service

## Tech Stack
- Frontend: React.js (Netlify)
- Backend: Node.js + Express (Railway)
- Database: MongoDB

## Live URLs
- Frontend: https://astounding-genie-e4914f.netlify.app
- Backend: https://YOUR-RAILWAY-URL.up.railway.app

## Run Locally
cd back-end-redbus && npm install && npm start
cd front-end-redbus && npm install && npm start
